# Retro-World_Aventure

## About
Retro World Aventure is a 2D platformer game featuring a purple bot character who shoots enemies, collect gems and coins, and unclock diffferent levels accross a dark fantasy world.

## Gameplay Features
- Shoot enemies with bullet
- Collect gems, coins and diamonds
- unlock doors with keys
- Health system with heart sprites

## Controls(QWERTY Keyboard)
movement: arrow keys or WASD
jump : Space bar
shoot: F
shoot_up: f + up arrow key
shoot_crouch: f + down arrow key
cling on a wall: jump + c ( + f if you want to shoot while on the wall)

## Levels
- Test level: tutorial/ practice area
- level 1 to level 6
  
## Build With
- Godot 4.6

## AI Usage
- Ai was used in this project for mainly debugging code erros and typos. All codes was written by me, following youtube tutorials and adapting it to fit my own game design decisions and features.
